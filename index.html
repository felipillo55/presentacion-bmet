Proceso de Reparación

Cuatro pasos fundamentales para restablecer el funcionamiento

1. Diagnosticar

Paso más complejo y susceptible a errores

  • Requiere conocimiento profundo del equipo
  • Nivel más complejo de pensamiento
  • Análisis del funcionamiento correcto

2. Localizar

Delimitar el problema a un área específica

  • Reducir el problema a una sección
  • Identificar área particular
  • Uso de equipos de prueba

3. Aislar

Identificar circuito o componente exacto

  • Localizar componente específico
  • Realizar mediciones precisas
  • Aplicación de análisis técnico

4. Determinar

Sustituir y verificar funcionamiento

  • Sustitución del componente defectuoso
  • Verificación completa del sistema
  • Pruebas de funcionamiento

Calibración y Mantenimiento

Garantizando el correcto funcionamiento de equipos

Calibración

Comparación con un estándar para garantizar funcionamiento preciso según parámetros preestablecidos.

Conceptos clave:
Estándar: Referencia establecida para determinar valores
Confiabilidad: Compatibilidad entre medida y valor real

Mantenimiento Preventivo

Revisión estructurada, periódica y regulada para garantizar óptimas condiciones.

Características:
Estructurado Periódico Documentado Regulado
Importante: Requerido en acreditaciones hospitalarias

Equipos de Prueba

Instrumentos esenciales para el trabajo del BMET

Multímetros

Medición de parámetros eléctricos

  • Voltaje
  • Corriente
  • Resistencia
  • Continuidad
  • Capacitancia

Osciloscopios

Visualización de señales

  • Cambios de voltaje
  • Amplitud
  • Periodo
  • Frecuencia
  • Fase

Generadores

Producción de señales

  • Ondas senoidales
  • Ondas cuadradas
  • Ondas triangulares
  • Parámetros variables

Simuladores

Señales médicas específicas

  • Actividad cardíaca
  • Presión sanguínea
  • Señales fisiológicas
  • Parámetros vitales
Nota: Una señal es cualquier cantidad que varía con el tiempo y conlleva información sobre un fenómeno.

Tipos de Manuales

Documentación esencial para el trabajo técnico

Manual del Usuario

Información general de funcionamiento del equipo

Contenido:
  • Funcionamiento general
  • Especificaciones técnicas
  • Diagramas generales
  • Teoría de funcionamiento
  • Procedimientos de calibración
Especificación: Descripción o parámetro de funcionamiento

Manual Técnico

Información especializada para técnicos

Contenido:
  • Información técnica específica
  • Funcionamiento detallado
  • Procesos de ajuste/calibración
  • Procedimientos de reparación (troubleshooting)
  • Diagramas técnicos completos
Nivel avanzado para técnicos especializados

Tipos de Diagramas

Representaciones visuales para comprender los equipos

Diagrama en Bloque

Funcionamiento general mediante rectángulos conectados

Características:
  • Rectángulos con funciones
  • Conexiones con líneas/flechas
  • Vista general del sistema
Uso:
  • Idea general del funcionamiento
  • Relación estructural

Diagrama Pictórico

Ilustración fiel de componentes físicos

Códigos: C (capacitor), R (resistencia), Q (transistor), U (CI)

Ventaja: Localización física de componentes

Diagrama Esquemático

Conexiones con símbolos estándar

Incluye:
  • Símbolos estándar
  • Nomenclatura
  • Valores de componentes
  • Conexiones detalladas
